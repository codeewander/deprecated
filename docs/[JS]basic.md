## Statement v.s. Expression

- Statement 敘述式: 沒有回傳值
  例如： if...else、switch、for 迴圈等
- Expression 表達式、運算式：有回傳值
  常見的例如：
  ```javascript
  //賦值運算式
  x = 1
  //比較運算式
  x == 1 or x === 1
  //算數運算式
  x + 1
  //邏輯運算式
  x && y
  ```

> [Expression @MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators)
